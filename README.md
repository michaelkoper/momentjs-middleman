# momentjs-middleman

Moment for Middleman!

## Installation

Add this line to your application's Gemfile:

    gem 'momentjs-middleman'

And then execute:

    $ bundle

## Usage

Add these lines to your javascript:

```js
//= require moment
```

You can also add location files:

```js
//= require moment/<locale>.js
```
